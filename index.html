BASICS OF WEB AND HTML NOTES: BUDRESKI

Intro: The web is made up of HTML, URLS, HTTP, Web apps
- Browsers send requests via the internet and http to servers which host the files and send them back to the browser for the user
- Just about any file can be found on the web

HTML = Hypertext markup lanaguage
- composed of: text content, markup, ref. to other docs
- plain text = plain text
- to change the text you need markup

Markup:
Tags - tags have an opening tag and a closing tag
ex: < name > contents < /name > and the whole string is an element
Bold tag = < b> content < /b>
Italics = < em> content < /em>
Note: If you don't close out a tag then the rest of the content on the page will be with that tag

Attributes in HTML:
Will look like: < Tag ATTR="value">contents< /tag>

- Anchor tag - < a> < /a>
- Used for making links on a webpage example: < a href="www.kdhgfihd.com">HTML< /a>

-Image tag Image tag - < img>
- Considered a void tag, meaning you don't have to close it example: < img src="imgage url" alt="text">

Note: You need to have alt=text in the case the image fails to load

- Break tag - < br> - also a void tag
- Makes text move down to the next line
- Only need to put it once right after where you want the next line to begin

-Paragraph tag - < p> < /p>
- Keeps content in a specified box

Note: a < br> tag is an inline and a < p> tag makes a box which you can set height and width of the box

Elements:
- Elements contain their content. You can attach styles to them via CSS
< span> < /span> are inline
< div> < /div> are block

HTML DOCUMENTS: What it looks like
